Portal Eleições BR 2026

Da Ideia à Concepção com IA

Caso de Uso — Documentação Técnica · Carlos Duplar · Março 2026

1
Visão Geral

O Portal Eleições BR 2026 é um sistema de monitoramento bilíngue (pt-BR + en-US) em tempo quase real das notícias relativas às eleições presidenciais brasileiras de 2026. O projeto agrega mais de 30 fontes entre veículos jornalísticos, sites partidários, institutos de pesquisa e redes sociais.

2
Motivação

Com o 1.º turno marcado para 4 de outubro de 2026 e mais de 150 milhões de eleitores, o ecossistema de informação eleitoral está fragmentado. Não existe plataforma centralizada, imparcial e atualizada.

3
Jornada de Concepção

O projeto nasceu de conversas estruturadas com Claude Sonnet 4.6 no Claude.ai e evoluiu iterativamente até um prompt de especificação técnica completo para Claude Opus 4.6 + GPT-5.3-Codex.

4
Arquitetura de IA

Cadeia de fallback hierárquica com 5 provedores de IA, todos usando OpenAI Python SDK com base_url swap. Zero lock-in, custo predominantemente gratuito.

5
Tech Stack

React + Vite com SSG. Pipeline Python. GitHub Actions CI/CD a cada 10 min. GitHub Pages hospedagem estática. Cloudflare CDN.

6
Lições Aprendidas

A combinação de vibe coding com IA (Opus planner + Codex implementador) permitiu produtividade equivalente a 3-4 desenvolvedores. Custo total abaixo de $15/mês.
